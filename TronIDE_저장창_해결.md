# TronIDE .sol 저장 — 해결

## Remixd (저장 가능, 추천)

**상세:** `Remixd_로컬폴더_연결_방법.md`

1. `npm install -g @remix-project/remixd`
2. `remixd -s C:\Users\hosuk\link-directory -u https://www.tronide.io` (터미널에서 실행, 유지)
3. TronIDE → REMIXD Activate → Connect to local Plugin → Connect
4. localhost에서 LinkUSD_TRON.sol 열기 → **Ctrl+S** = 로컬 저장됨

---

## Load a local file (저장 불가)

Load a local file → LinkUSD_TRON.sol 선택. 수정 불가. Compile·Deploy만.
