## 블록 처리 / "지워진" 것

| 상황 | 설명 |
|------|------|
| **실패한 거래** | 블록에는 **기록됨** (해시 있음). 다만 **실행은 실패** → USDT 전송/승인 안 됨. |
| **지워진 거래** | 블록체인은 수정 불가. 거래가 "삭제"되지는 않음. |
| **에너지 부족** | 트랜잭션은 제출됐지만, 에너지가 부족해 **실행 단계에서 실패**한 것. |
