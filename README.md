# Green Point (GRP) — TRON TRC20 토큰

TRON 블록체인 기반 TRC20 토큰. Sun Pump 론치 + TronScan 배포. 온체인 Burn, Lock 기능.

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **토큰 이름** | Green Point |
| **심볼** | GRP |
| **표준** | TRC-20 |
| **블록체인** | TRON (Mainnet / Nile Testnet) |
| **초기 공급량** | 1,000,000,000 (10억) |
| **decimals** | 6 |
| **라이선스** | MIT |

---

## 주요 기능

- **transfer, approve, transferFrom** — TRC20 기본
- **burn** — 사용자 직접 온체인 소각
- **lock / unlock** — Owner 권한으로 계정 잠금·해제
- **transferOwnership** — 소유권 이전
- **checkLocked** — 잠금 상태 조회
