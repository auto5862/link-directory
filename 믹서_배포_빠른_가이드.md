# 믹서 배포 — 빠른 가이드

**MultiTokenMixer: 공식 USDT + USDT(TeatherUSD) 같이 믹싱**

---

## 1. 준비물

| 항목 | 값 |
|------|-----|
| TRX | ~80~100 개 (Energy 소모) |
| TronLink | 연결 |
| 파일 | MultiTokenMixer.sol |

---

## 2. 배포 주소 (constructor)

| 파라미터 | 주소 |
|----------|------|
| **_token1** | `TR7NHqjeKQxGTCi8q8ZY4pL8otSzgjLj6t` (공식 USDT) |
| **_token2** | `TGacvY1cTSLKgWEPxXqWt8FFAQ86vpzM1U` (USDT TeatherUSD) |

---

## 3. TronIDE에서 배포

### 3-1. 접속
- https://www.tronide.io 접속
- **Mainnet** 선택

### 3-2. 코드 넣기
1. **+** (새 파일) → `MultiTokenMixer.sol` 생성
2. 프로젝트의 `MultiTokenMixer.sol` 내용 **전부** 복사 → 붙여넣기
3. 저장

### 3-3. Deploy
1. **Compile** 클릭 (에러 없으면 OK)
2. **Deploy** 탭
3. **CONTRACT** → `MultiTokenMixer` 선택
4. **constructor** 입력:
   - `_token1`: `TR7NHqjeKQxGTCi8q8ZY4pL8otSzgjLj6t`
   - `_token2`: `TGacvY1cTSLKgWEPxXqWt8FFAQ86vpzM1U`
5. **Deploy** 클릭
6. TronLink → **Confirm** (TRX 결제)

### 3-4. 완료
- 배포된 **믹서 주소**: `TC812jp3UnGKgjY4aZCvjNWre7b3YqXJVB`

---

## 4. 사용 (배포 후)

### 예치 (deposit)
1. 토큰 **approve**(믹서주소, 1000000000) — 1000 * 10^6
2. **deposit**(토큰주소, 비밀해시) 호출

### 출금 (withdraw)
1. **withdraw**(토큰주소, 비밀값, 수신주소) 호출
