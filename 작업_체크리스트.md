## 1단계: Link USD 배포 (지금 할 것)

**⚠️ CONTRACT: LinkUSD 선택 (TetherUSD 아님 — TetherUSD 선택 시 사기 표시됨)**

### 1-1. TronIDE 접속
- https://www.tronide.io/
- TronLink 연결 (Connect)

### 1-2. 코드 확인
- 왼쪽 **1_Storage.sol** 더블클릭
- **`contract LinkUSD`**, `burn`, `lock` 있는지 확인 (TetherUSD 아님)
- 없으면 **LinkUSD_TRON.sol** 전체 붙여넣기
- **Ctrl+S** 저장

### 1-3. 배포
- 왼쪽 **번개 아이콘(⚡)** 클릭
- ENVIRONMENT: **Injected TronWeb** 선택
- CONTRACT: **LinkUSD** 선택 (TetherUSD 절대 금지)
- **Deploy** 클릭
- TronLink에서 **Sign** 클릭

### 1-4. 배포 후
- Deployed Contracts에서 **컨트랙트 주소 복사** (T로 시작)
- TronLink → Custom Token → 주소 붙여넣기 → 토큰 추가

## 1-5. 토큰 로고 등록 (레포트용 - USDT처럼 보이게)

**※ 기본 아이콘(노란 별) 대신 로고가 보이게 하려면 반드시 진행**

### ⚠️ 주의: "편집하다" 버튼은 아님!
- 컨트랙트 페이지의 **편집하다** → 에너지 소비 비율만 수정 (로고 아님!)
- 로고는 **Wallet → Record a Token** 경로에서 수정

### 절차 (이미 등록된 토큰 정보 수정)
1. TronScan **우측 상단** 지갑 주소(`TVMfvZx...`) 클릭 (메인넷: tronscan.org)
2. 드롭다운에서 **Wallet** 또는 **Record a Token** 메뉴 찾기
3. 없으면 **직접 URL** 시도:
   - https://tronscan.org/#/tokens/create/Record (메인넷)  
   - TRC20 선택 → 주소 입력 → "이미 등록됨" 시 나오는 안내 링크 따라가기
4. Logo (imgUrl) 입력: `https://static.tronscan.org/production/logo/usdtlogo.png`
5. Submit

### Nile 테스트넷 한계
- Nile은 메인넷보다 UI가 제한적일 수 있음
- 레포트용: 로고 미표시도 "테스트넷 환경 특성"으로 설명 가능

## 2단계: 전송 테스트 (Nile에서 먼저!)

**→ 상세: `전송_테스트_가이드.md` 참고**

### 요약
1. **받을 주소**: TronLink → 설정 → 계정 추가 → 주소 복사
2. **Send (⬆️)** 클릭 → 주소 칸에 **붙여넣기** (No transfer record = 정상, 직접 입력)
3. **다음** → 수량 **1000000** (= 1 USDT) → **Sign**

---

## 3단계: Burn 테스트

1. TronIDE → Deployed Contracts → LinkUSD (또는 배포한 contract 이름)
2. **burn** 함수 찾기
3. amount 입력 (예: 1000000 = 1 USDT, decimals 6)
4. transact 클릭 → Sign
5. **스크린샷 찍기**

---

## 4단계: Lock 테스트

1. TronIDE → LinkUSD 컨트랙트 (또는 배포한 contract 이름)
2. **lock** 함수 → 테스트할 주소 입력
3. transact → Sign
4. 해당 주소로 전송 시도 → **차단되는지 확인**
5. **unlock** 함수로 해제
6. **스크린샷 찍기**

---

## 5단계: TRX 더 있으면 - USDTMixer 배포

### 5-1. TronIDE에 믹서 코드 넣기
- 새 파일 또는 2_Owner.sol 열기
- USDTMixer.sol 내용 전체 붙여넣기
- **constructor에 토큰 주소 필요!** (1단계에서 배포한 LinkUSD 주소)
- Ctrl+S 저장

### 5-2. 배포
- Deploy → Injected TronWeb
- CONTRACT: **USDTMixer** 선택
- constructor: **_tokenAddress**에 LinkUSD 컨트랙트 주소 입력
- Deploy → Sign

## 6단계: TRX 더 있으면 - SunSwap 유동성 풀 (시세 반영 구현)

### 6-1. 시세 조회 (실제 구현!)
1. https://coinmarketcap.com 또는 https://www.coingecko.com 접속
2. **TRX** 검색 → 현재 가격 확인 (예: $0.28)
3. **USDT** = $1.00 (고정)

### 6-2. 비율 계산
```
1 USDT = $1.00
1 TRX = $0.28 (당일 시세)

비율: 1 USDT = 1 ÷ 0.28 = 3.57 TRX
```

### 6-3. 입금량 결정
예: USDT 10,000개 넣을 경우
- TRX = 10,000 × 3.57 = **35,700 TRX**

### 6-4. SunSwap 실행
1. https://sunswap.com 접속
2. **Add Liquidity** 클릭
3. 토큰: **TRX** + **니 USDT** (컨트랙트 주소로 추가)
4. **6-3에서 계산한 수량** 입력
5. 트랜잭션 승인

### 6-5. 레포트에 기입
- 5.3절: 실제 적용한 시세, 비율, 입금량 작성
- 6.1절: 스왑 테스트 결과 기입

## ★ 메인넷 배포 (실전)

### M-1. 사전 준비
| 항목 | 내용 |
|------|------|
| TRX 필요량 | **최소 350 TRX** (배포 수수료) |
| 네트워크 | TronLink → **Mainnet**으로 변경 |

### M-2. TronLink 메인넷 전환
1. TronLink 확장 프로그램 클릭
2. 좌측 하단 또는 설정에서 **네트워크** 선택
3. **Mainnet** (메인넷) 선택
4. 메인넷 TRX 잔액 확인 (350 이상 필요)

### M-3. TronIDE에서 메인넷 배포
1. https://www.tronide.io/ 접속
2. TronLink 연결 (**Mainnet** 상태인지 확인!)
3. LinkUSD_TRON.sol 코드 확인 (1_Storage.sol에 붙여넣기)
4. **Deploy** → Injected TronWeb → **LinkUSD** (TetherUSD 절대 금지)
5. **Sign** → 배포 완료
6. **컨트랙트 주소 복사** (T로 시작)

### M-4. TronLink에 토큰 추가
- Custom Token → 메인넷 배포한 컨트랙트 주소 입력

### M-5. 메인넷 TronScan 로고 등록
- **tronscan.org** (nile 아님!) 접속
- Record Token → TRC20 → 컨트랙트 주소 입력
- **Logo (imgUrl)** 필드에 `https://static.tronscan.org/production/logo/usdtlogo.png` 입력
- Submit (메인넷에는 로고 필드 있음)

### M-6. 피싱 경고 피하려면
- **CONTRACT: LinkUSD** 선택 (TetherUSD 선택 시 사기 표시)
- name = "Link USD", symbol = "LUSD" (`Contract_Name_명시.md` 참고)
