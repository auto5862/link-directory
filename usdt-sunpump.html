전송 방법 (차근차근)

  1. 어디로: 이 페이지 (usdt-sunpump.html) 또는 GitHub Pages
  2. 준비: Chrome + TronLink 설치. 지갑에 USDT(TeatherUSD) + TRX(수수료용) 보유
  3. 1단계: "지갑 연결 & 잔액 확인" 클릭 → TronLink 팝업에서 연결 클릭
  4. 2단계: "받을 주소"에 상대 공식 USDT 주소 입력 (T로 시작, 예: TXXX...)
  5. 3단계: "금액"에 보낼 USDT 입력 (예: 1)
  6. 4단계: "전송" 버튼 클릭
  7. 5단계: TronLink 팝업 뜨면 승인(Confirm) 클릭
  8. 6단계: 받는 사람에게 receive-usdt.html 링크 전달 → 버튼 한 번 클릭으로 지갑에 표시
  9. 실패 시: "TronScan Contract Write" 링크 → transfer(to, amount) 직접 호출

트랜잭션 옵션 (선택)

구성 가능 항목:

  • feeLimit: 최대 수수료(SUN). 1 TRX=1,000,000 SUN. 기본 100 TRX.
  • callValue: 컨트랙트에 보낼 TRX(SUN). TRC20 전송은 0.

전송: triggerSmartContract로 직접 트랜잭션 구성 (오류 없음)

모든 방법 실패 시: TronScan Contract Write에서 직접 transfer 호출

사용 가이드

Burn · Lock

USDT (TeatherUSD)는 Sun Pump 표준 TRC-20. transfer, approve만 지원. Burn/Lock 없음.

Burn/Lock이 필요한 경우: LUSD (Link USD) 컨트랙트 사용. tether-tools.html 또는 TronScan → Contract → Write에서 burn(), lock() 호출.

탈중앙화 & 반테더

테더(Tether)는 회사가 발행·예비자산을 통제하는 중앙화 구조. USDT (TeatherUSD)는 온체인 발행·검증, DEX 거래로 탈중앙화 지향.
