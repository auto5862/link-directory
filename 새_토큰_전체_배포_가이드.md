# 1. 배포 전 (공통)

## 1-1. TronLink

| # | 확인 | 값 |
|---|------|-----|
| 1 | 네트워크 | Nile 또는 Mainnet |
| 2 | Nile RPC | `https://nile.trongrid.io` / Chain ID `3` |
| 3 | Nile TRX | nileex.io → 주소 입력 → TRX 2000 받기 |

## 1-2. 지갑 태그 검증 (배포 전 필수)

**Nile:** nile.tronscan.org/#/tools/verify-sign  
**Mainnet:** tronscan.org/#/tools/verify-sign

| 단계 | 어디 | 뭘 함 |
|------|------|-------|
| 1 | **Sign Message** | 메시지 `I confirm this address belongs to Green Point.` (또는 Alpha Point이면 `I confirm this address belongs to Alpha Point.`) → 서명 → **서명 해시 + 원본 메시지** 복사 |
| 2 | **문의하기** → **태그 제출** | 위에서 복사한 내용 붙여넣기 → 제출 |

**⚠️ Short Description 맨 위에 서명 해시 + 원본 메시지 붙여넣기**

**태그 제출 기입란:**

| 기입란 | Green Point | Alpha Point |
|--------|-------------|-------------|
| Address | 발행자 지갑 주소 | 발행자 지갑 주소 |
| Tag | Green Point | Alpha Point |
| Short Description | Green Point (GRP) — TRON TRC20. Burn·Lock. | Alpha Point (AP) — TRON TRC20. Burn·Lock. |
| Project Official Website | 프로젝트 URL | 프로젝트 URL |
| Project Name | Green Point | Alpha Point |
| Contact Info | 이메일 | 이메일 |

## 1-3. 코드 확인

```
contract GreenPoint {           ← 또는 AlphaPoint
    string public name = "Green Point";   ← 또는 "Alpha Point"
    string public symbol = "GRP";     ← 또는 "AP"
```

**금지:** USD, USDT, Tether, Link, Blue (기존 프로젝트와 혼동되는 이름)
