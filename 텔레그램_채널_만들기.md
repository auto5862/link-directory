# 텔레그램 채널 만들기 — 전부

**TronScan 토큰 평판 업데이트 때 "소셜"로 제출할 거.**

---

## 0. 준비물 — 채널 이미지 (프로필 사진)

**채널 개설 시 이미지 필수.**

| 항목 | 내용 |
|------|------|
| **용도** | 채널 프로필 사진 (채널 대표 이미지) |
| **규격** | 256×256 PNG 권장 (정사각형) |
| **내용** | Link USD 로고 (LUSD 아이콘 등) |
| **재사용** | TronScan Record Token 로고랑 같은 이미지 써도 됨 |

→ 0단계 준비물에 **로고 이미지 256×256 PNG** 있음. 그거 쓰면 됨.

---

## 1. 채널 vs 그룹 — 뭘 만들까?

| 구분 | 채널 | 그룹 |
|------|------|------|
| **용도** | 공지·발표 (일방향) | 대화·질문 (양방향) |
| **추천** | ✅ **채널** (프로젝트 공식용) | 그룹도 가능 |
| **t.me 링크** | 둘 다 가능 | 둘 다 가능 |

→ **채널** 만들면 됨. 나중에 그룹 추가해도 됨.

## 3. 사용자 이름 (t.me/뒤에 붙는 거)

- **예:** `linkusd` → URL은 `https://t.me/linkusd`
- 이미 쓰인 이름이면 `linkusd_official`, `linkusd_lusd` 등으로 변경

---

## 4. 주제·설명 (채널 소개)

**아래 중 하나 복사해서 채널 설명에 넣기:**

```
Link USD (LUSD) — TRON TRC20 스테이블코인. 온체인 Burn·Lock·투명성.
웹: https://auto5862.github.io/link-directory/
백서: https://auto5862.github.io/link-directory/TetherUSD_Whitepaper.md
```

→ `auto5862`를 본인 GitHub 유저명으로 바꾸기.

---

## 5. 만드는 순서

1. **로고 이미지** 준비 (256×256 PNG, Link USD 로고)
2. **Telegram** 앱/PC 열기
3. **메뉴** → **새 채널** (New Channel)
4. **채널 이미지** 업로드 (프로필 사진 — 위에서 준비한 로고)
5. **채널 이름**: `Link USD` (또는 위 2번에서 고른 제목)
6. **설명**: 위 4번 내용 붙여넣기
7. **채널 유형**: **공개 채널** 선택
8. **링크**: `linkusd` (또는 `linkusd_official` 등) 입력
9. 완료 후 **설정** → **채널 정보** → **설명**에 웹·백서 URL 있는지 확인

---

## 6. 프로필에 넣을 링크

채널 **설명**에 아래 넣기:

- 웹사이트: `https://[유저명].github.io/link-directory/`
- 백서: `https://[유저명].github.io/link-directory/TetherUSD_Whitepaper.md`

## 7. 적어둘 URL

만들고 나서 **채널 링크** 복사:

```
https://t.me/linkusd
```

→ `배포_전_할_일_순서.md` 2번 `_______________________` 칸에 적어두기

## 9. 채널에 유저 모으는 법

| 방법 | 설명 |
|------|------|
| **웹·백서에 링크** | link-usd.html, index.html, GitHub README에 t.me/whywhyme1 링크 넣기 |
| **TronScan 토큰 페이지** | 토큰 평판 업데이트 후 소셜 링크 노출 → 클릭 시 채널로 이동 |
| **트위터/X** | Link USD 관련 글 + 채널 링크 트윗 |
| **TRON 커뮤니티** | TRON 포럼, 디스코드, 텔레그램 그룹에서 프로젝트 소개 + 채널 링크 |
| **DEX 상장 후** | SunSwap 등 상장 시 토큰 페이지에 웹·소셜 링크 등록 |
| **꾸준한 공지** | 채널에 업데이트·진행 상황 올리면 검색·공유 시 노출됨 |

→ **먼저**: 웹사이트·백서·TronScan에 채널 링크 걸어두는 게 기본.
