# 메인넷 두 번째 주소 찾기

**결제 2번 = 계약 2개. 다른 주소 찾아서 처음부터 안 뜨게.**

---

## 1. 찾는 방법

### tronscan.org에서

1. **tronscan.org** 접속 (메인넷)
2. 검색창에 **발행자 주소** 입력: `TVMfvZxNrTWE9jFoN4RBdyercKhLpRYrFm`
3. **Transactions** 탭 → **Filter** 또는 **유형**에서 **"Contract Creation"** 또는 **"스마트 계약 생성"** 선택
4. **스마트 계약을 생성했습니다** 트랜잭션들 확인
5. 각 트랜잭션 클릭 → **연락처(Contract Address)** 확인

### 또는

1. tronscan.org → 주소 `TVMfvZxNrTWE9jFoN4RBdyercKhLpRYrFm` 검색
2. **Contracts** 탭 (있으면) → 생성한 계약 목록
3. 또는 **Transactions** → 날짜순으로 스크롤 → "Contract Creation" 찾기
