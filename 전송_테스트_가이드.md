# USDT 전송 테스트 - 한 번에 정리

---

## 1. 받을 주소 만들기 (없으면)

1. TronLink 클릭 → **설정(⚙️)** → **계정 관리**
2. **계정 추가** 또는 **새 계정 생성**
3. 새 계정 선택 → **주소 복사** (T로 시작하는 34자)

---

## 2. 전송 (5단계)

| 순서 | 화면 | 할 일 |
|------|------|-------|
| 1 | TronLink 메인 | **위쪽 화살표(⬆️)** 클릭 = Send |
| 2 | Send (1/2) | **"Paste or enter the account address"** 칸에 받을 주소 **붙여넣기** |
| 3 | Send (1/2) | **다음** 또는 **Next** 클릭 |
| 4 | Send (2/2) | 수량 입력: **1000000** (= 1 USDT) |
| 5 | 확인 | **확인** → **Sign** |

---

## 3. "No transfer record" 의미

- **에러 아님** → 아직 보낸 적 없어서 내역이 비어 있는 것
- **그냥 주소 칸에 직접 입력/붙여넣기** 하면 됨

---

## 4. 수량 참고 (decimals 6)

| 입력 | 의미 |
|------|------|
| 1000000 | 1 USDT |
| 10000000 | 10 USDT |
| 100000000 | 100 USDT |

---

## 5. 확인

- 받는 지갑에서 USDT 잔액 증가 확인
- Nile TronScan: https://nile.tronscan.org 에서 트랜잭션 검색
