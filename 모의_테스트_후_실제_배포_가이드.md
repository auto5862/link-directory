# 모의 테스트 → 실제 배포 가이드

**TronLink 먼저 테스트** → Base Sepolia → Base 메인넷.

---

## 전체 순서

| 단계 | 네트워크 | 지갑 | 목적 | 비용 |
|------|----------|------|------|------|
| **1** | TRON Nile (테스트넷) | **TronLink** | TRON 배포·전송 연습 | 무료 |
| **2** | Base Sepolia (테스트넷) | MetaMask | Uniswap 연습 | 무료 |
| **3** | Base 메인넷 | MetaMask | 실제 배포 | ETH 필요 |

---

# 1단계: TRON Nile 모의 테스트

## 1-1. Nile TRX 받기 (무료)

1. TronLink → 네트워크 **Nile** 선택
2. **https://nileex.io/join/getJoinPage** 접속
3. TronLink 주소 입력 → CAPTCHA → 제출
4. **2000 TRX** 수령 (24시간당 1회)

## 1-2. Nile에서 토큰 배포

1. **TronIDE** (https://www.tronide.io) 또는 **Remix** (TRON 플러그인)
2. `LinkUSD_TRON.sol` (contract LinkUSD) 사용
3. TronLink 연결 → 네트워크 **Nile** 선택
4. Deploy → 컨트랙트 주소 저장

## 1-3. Nile에서 전송 테스트

1. TronLink에서 방금 배포한 토큰 추가
2. **전송** → 다른 주소로 보내기
3. Nile TronScan (nile.tronscan.org)에서 확인

## 1-4. Nile SunSwap (있으면)

- Nile에 SunSwap 테스트넷이 있으면 유동성 풀 연습
- 없으면 1-3까지로 TRON 흐름 확인 완료

**완료 확인:** Nile에서 배포 → 전송까지 문제없이 됨.

---

# 2단계: Base Sepolia 모의 테스트 (MetaMask 연습)

## 2-1. MetaMask에 Base Sepolia 추가

1. **https://chainlist.org** 접속
2. **"테스트넷을 포함하세요"** 체크
3. 검색: **"Base Sepolia"**
4. **메타마스크에 추가** 클릭

## 2-2. Sepolia ETH 받기 (무료)

1. **https://www.alchemy.com/faucets/base-sepolia** 또는
2. **https://faucet.quicknode.com/base/sepolia**
3. MetaMask 주소 입력 → ETH 수령

## 2-3. Remix에서 Base Sepolia 배포

1. **https://remix.ethereum.org** 접속
2. `LinkUSD_Ethereum.sol` 배포
3. MetaMask → **Base Sepolia** 네트워크 선택
4. Deploy → 컨트랙트 주소 저장

## 2-4. Uniswap에서 유동성 풀 연습

1. **https://app.uniswap.org** 접속
2. MetaMask → **Base Sepolia** 선택
3. Pool → New position
4. **Select a token** → 배포한 컨트랙트 주소 붙여넣기 → **Import**
5. ETH + 토큰 금액 입력 → Add

**완료 확인:** Base Sepolia에서 배포 → Uniswap Import → 유동성 추가까지 됨.

---

# 3단계: Base 메인넷 실제 배포

**2단계까지 문제없이 됐으면** → `이더리움_배포_Uniswap_가이드.md` 따라 진행.

1. MetaMask에 **Base 메인넷** 추가 (chainlist.org)
2. 거래소에서 **ETH 0.01** 구매 → Base로 출금
3. Remix에서 **Base 메인넷**으로 배포
4. Uniswap에서 **Base** 네트워크로 유동성 추가

## 요약

1. **Nile** → TRX 무료 받아서 배포·전송 연습
2. **Base Sepolia** → ETH 무료 받아서 MetaMask + Uniswap 연습
3. **Base 메인넷** → 실제 ETH 넣고 배포

**테스트넷에서 막히면** → 메인넷 가기 전에 해결.
