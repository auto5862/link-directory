# 된다고 확인한 기능 기록

---

## 1. LinkUSD_TRON.sol 컨트랙트 기능

| 기능 | 함수 | 상태 |
|------|------|------|
| 개인지갑간 전송 | `transfer()` | 된다 |
| 락 | `lock()` | 된다 |
| 락 해제 | `unlock()` | 된다 |
| 소각(Burn) | `burn()` | 된다 |
| 위임 전송 | `approve()` + `transferFrom()` | 된다 |
| 소유권 이전 | `transferOwnership()` | 된다 |
| 잠금 상태 확인 | `checkLocked()` | 된다 |

---

## 2. 잔액 표시

| 항목 | 상태 |
|------|------|
| 내 지갑에 토큰 수량(금액) 표시 | 된다 |
| 받는 상대 지갑에 토큰 수량(금액) 표시 | 된다 |
| 현재가 기준 달러($) 표시 | TronLink가 처리. 미상장 시 $0, SunSwap 상장 시 시세 반영 |

---

## 3. 메인넷 관련

| 항목 | 상태 |
|------|------|
| TronScan 로고 등록 | tronscan.org → Record Token → Logo (imgUrl) `https://static.tronscan.org/production/logo/usdtlogo.png` 입력 → 된다 |
| PC-폰 지갑 연동 | 같은 시드(Mnemonic) 사용 → 된다 |
| 폰 TronLink | Add Wallet → Import Wallet → Mnemonic 입력 |
| 메인넷 네트워크 | 폰 TronLink에서 Mainnet 선택해야 메인넷 잔액 표시 |

---

## 4. PC TronLink 시드 구문 내보내기

| 경로 | 내용 |
|------|------|
| Wallet Details | 클릭 |
| Export Wallet → Mnemonic | 시드 구문(12단어) 확인 |

---

## 5. 폰 TronLink 지갑 가져오기

| 경로 | 내용 |
|------|------|
| Add Wallet | 지갑 추가 |
| Import Wallet | 지갑 가져오기 |
| Mnemonic | PC에서 확인한 12단어 입력 |
| Name | 지갑 별칭 (아무거나 입력 가능) |

---

## 6. 받는 사람 확인 방법 (token-receive.html)

| 방법 | 내용 |
|------|------|
| ① 지갑 연결 | TronLink 연결 → 버튼 클릭 → 이 페이지에서 잔액 확인 (토큰 추가 불필요) |
| ② TronScan | 주소 입력 → Nile/Mainnet 링크 클릭 → 모든 TRC20 내역 자동 표시 |
| ③ 토큰 추가 | 버튼 한 번 → TronLink에 표시 |
| ④ CMC 등록 | 장기: SunSwap 상장 후 CoinMarketCap 신청 |

**송금 시 받는 사람에게 전달:** `tether-tools.html` 링크 (한 페이지에 모든 도구 모음) 또는 token-receive.html

## 7. 우리 USDT → TRX → XRP/ETH 절차

| 단계 | 내용 |
|------|------|
| 1 | SunSwap 유동성 풀 (우리 USDT + TRX) 생성 |
| 2 | 상대: token-receive.html로 받기 확인 |
| 3 | 상대: SunSwap에서 우리 USDT → TRX 스왑 |
| 4 | 상대: TRX → 거래소 입금 → XRP/ETH 교환 |

**상세 절차:** `스왑_XRP_ETH_절차.md`
