## ChainList "Base RPC URL List"가 뭔가요?

**Base 네트워크 RPC 서버 목록**입니다. MetaMask가 Base 블록체인에 연결할 때 쓰는 서버 주소들.

- **하는 일:** Base 네트워크 설정 (이미 추가했으면 건드릴 필요 없음)
- **TRX·USDT와 무관:** TRON이 아니라 Base(이더리움 L2)용

## TRON (TronLink) — 50 USDT 묶여 있음

**상황:** TRX 없음 → USDT 이체 불가

**해결:** TRX만 받으면 이체 가능

### 무료 TRX (보증금 없어야 함)

| 사이트 | 링크 |
|--------|------|
| TRX True | https://trxtrue.com |
| Free-Tron.com | https://free-tron.com |
| Tronpayu | https://tronpayu.io |
| Firefaucet | https://firefaucet.win (TRX 포함) |
| Faucetcrypto | https://faucetcrypto.com |
| Dutchycorp | https://dutchycorp.space |
| Clicktoshi | https://clicktoshi.com |

※ 보증금·선입금 요구하면 사용하지 말 것. 사이트마다 정책이 다를 수 있음.

**TronLink GasFree:** TRX 없이 USDT 전송 가능하지만, USDT를 GasFree 지갑으로 옮기려면 그 전송에 TRX 필요. 지금처럼 TRX 0이면 사용 불가.

### 방법: Energy 대여 (USDT로 결제)

**Tronex Energy** — USDT로 에너지 대여 후 TRX 없이 전송 가능.

**문제:** Tronex에 USDT를 입금하려면, 그 전송 자체에 TRX가 필요함. TRX 0이면 첫 입금 불가.

---

### 방법: 누군가에게 TRX "dust" 요청

- 지인, 가족에게 **TRX 10~20개** (약 $3~6) 요청
- TRON 텔레그램/레딧 커뮤니티에서 도움 요청 (사기 주의)
- 주소: `TVMfvZxNrTWE9jFoN4RBdyercKhLpRYrFm`

---

### 방법: Freecash (무료, 시간 걸림)

- **https://freecash.com** — 게임·설문 등으로 적립 → TRX로 출금
- 가입 보너스 $10
- 보증금 없음. 단, 1~2 TRX 모으려면 시간 필요

---

### 거래소 TRX 구매 (가장 빠름)

| 단계 | 할 일 |
|------|-------|
| 1 | 거래소(업비트, 바이낸스 등) 가입 또는 로그인 |
| 2 | 원화 입금 → TRX 매수 (10~50 TRX, 약 2~10달러) |
| 3 | 출금 → **네트워크: TRON** 선택 |
| 4 | 받는 주소: TronLink 지갑 주소 (TVMfvZxNrTWE9jFoN4RBdyercKhLpRYrFm) |
| 5 | 출금 완료 후 TronLink에서 USDT 이체 가능 |

**TRX 들어오면:** USDT를 다른 지갑이나 거래소로 보낼 수 있음.

## Base — ETH 없어서 진행 불가

**상황:** ETH 없음 → 배포·유동성 풀 불가

**해결:** ETH만 받으면 진행 가능

| 단계 | 할 일 |
|------|-------|
| 1 | 거래소에서 ETH 매수 (0.01 ETH 정도, 약 30~50달러) |
| 2 | 출금 → **네트워크: Base** 선택 (이더리움 메인넷 아님) |
| 3 | 받는 주소: MetaMask Base 주소 (0x7bC9...340fd3) |
| 4 | 출금 후 `이더리움_배포_Uniswap_가이드.md` 3단계부터 진행 |
