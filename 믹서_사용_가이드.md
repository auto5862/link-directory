# 믹서 사용 가이드

**예치 → 출금 (다른 주소로)**

---

## 1. 두 가지 방식

| 방식 | 설명 |
|------|------|
| **TokenMixer** | 토큰 1개만 (따로따로) |
| **MultiTokenMixer** | USDT + 우리 토큰 같이 섞여 있음 |

---

## 2. MultiTokenMixer (2번 — 같이 섞임)

### 배포 완료

| 항목 | 주소 |
|------|------|
| **믹서 주소** | `TC812jp3UnGKgjY4aZCvjNWre7b3YqXJVB` |
| _token1 | 공식 USDT `TR7NHqjeKQxGTCi8q8ZY4pL8otSzgjLj6t` |
| _token2 | USDT TeatherUSD `TGacvY1cTSLKgWEPxXqWt8FFAQ86vpzM1U` |

### 예치 (deposit)

1. 토큰 **approve** (믹서 주소, 1000 * 10^6)
2. **deposit(토큰주소, 비밀해시)** 호출
   - 비밀해시 = keccak256(abi.encodePacked(비밀값, 토큰주소))

### 출금 (withdraw)

1. **withdraw(토큰주소, 비밀값, 수신주소)** 호출
