# 믹서 사용 가이드

**예치 → 출금 (다른 주소로)**

---

## 1. 세 가지 방식

| 방식 | 설명 |
|------|------|
| **TokenMixer** | 토큰 1개 (USDT TeatherUSD decimals 18 전용) |
| **MultiTokenMixer** | 공식 USDT + USDT TeatherUSD 같이 (decimals 6 기준) |

---

## 2. TokenMixer (USDT TeatherUSD 전용 — decimals 18)

### 배포 방법

1. https://www.tronide.io → **Mainnet**
2. `TokenMixer.sol` 열기 → 코드 붙여넣기 (FIXED_AMOUNT = 1000 * 10^18)
3. **Compile** → **Deploy** → `_token`: `TGacvY1cTSLKgWEPxXqWt8FFAQ86vpzM1U`
4. TronLink **Confirm**

### 배포 완료

| 항목 | 주소 |
|------|------|
| **믹서 주소** | `TJRMpywqc8FH5ankFBgYjjmWW5modzqwbq` |
| _token | USDT TeatherUSD `TGacvY1cTSLKgWEPxXqWt8FFAQ86vpzM1U` |

### 예치 (deposit)

1. **approve** (믹서 주소 `TJRMpywqc8FH5ankFBgYjjmWW5modzqwbq`, `1000 * 10^18`)
2. **deposit(비밀해시)** 호출 — 비밀해시 = keccak256(abi.encodePacked(비밀값))

### 출금 (withdraw)

1. **withdraw(비밀값, 수신주소)** 호출

## 3. MultiTokenMixer (공식 USDT + USDT TeatherUSD — decimals 6만 정상)

### 배포 완료

| 항목 | 주소 |
|------|------|
| **믹서 주소** | `TC812jp3UnGKgjY4aZCvjNWre7b3YqXJVB` |
| _token1 | 공식 USDT `TR7NHqjeKQxGTCi8q8ZY4pL8otSzgjLj6t` |
| _token2 | USDT TeatherUSD `TGacvY1cTSLKgWEPxXqWt8FFAQ86vpzM1U` |

### 예치 (deposit)

1. 토큰 **approve** (믹서 주소, `1000 * 10^6`) — 공식 USDT만 정상
2. **deposit(토큰주소, 비밀해시)** 호출
   - 비밀해시 = keccak256(abi.encodePacked(비밀값, 토큰주소))

### 출금 (withdraw)

1. **withdraw(토큰주소, 비밀값, 수신주소)** 호출
