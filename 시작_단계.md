## 0단계: 지금 바로 (사전 준비)

1. **TRX 구매** — 거래소에서 TRX 구매 후 본인 지갑으로 전송
   - 최소 350 TRX (배포용)
   - 유동성 풀까지 하려면 수만 TRX 수준 (시세에 따라)
2. **TronLink** — 설치 후 **Mainnet** 선택
3. **백서** — TetherUSD_Whitepaper.md (이미 작성됨)

---

## 1단계: 배포 (TRX 350+ 있으면)

1. TronIDE 접속: https://www.tronide.io/
2. TronLink 연결, **Mainnet** 확인
3. 1_Storage.sol에 LinkUSD_TRON.sol 코드 붙여넣기 → 저장
4. Deploy → Injected TronWeb → **LinkUSD** 선택 (TetherUSD 아님) → Sign
5. **컨트랙트 주소 복사** (T로 시작)
6. TronLink → Custom Token → 주소 입력

---

## 2단계: TronScan 등록

1. tronscan.org 접속 (메인넷)
2. Tokens → Record Token → TRC20
3. 컨트랙트 주소 입력
4. Logo: `https://static.tronscan.org/production/logo/usdtlogo.png`
5. Verify Contract → 소스 업로드

---

## 3단계: SunSwap 유동성 풀

1. TRX 시세 조회 (coinmarketcap / coingecko)
2. 비율 계산: 1 USDT = 1÷TRX가격 TRX
3. SunSwap → Add Liquidity
4. 우리 USDT(컨트랙트 주소) + TRX 입력
5. Sign

---

## 4단계: token-receive.html

- 컨트랙트 주소 입력란을 **메인넷 배포 주소**로 변경

## 4-1. tether-tools.html (도구 모음)

- token-receive, add-token, tx-history, SunSwap 링크를 **한 페이지**에 모음
- 수신자에게 `tether-tools.html` 링크만 전달하면 됨
