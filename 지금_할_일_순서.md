### Tronscan 배포 전 (지갑 주소 공개 태그 검증)

1. **nile.tronscan.org**/#/tools/verify-sign 접속
2. Sign Message → TronLink 연결 → 메시지 `I confirm this address belongs to Link USD.` → Sign → **서명 해시** 복사
3. Verify Signature → 입력 후 Verify
4. **nile.tronscan.org**/#/tools/contactUs → **태그 제출** → 복사한 내용 붙여넣기 → 제출

**지갑 연결 안 되면:** Tronscan(nile.tronscan.org)과 TronLink(Nile) **네트워크 같아야 함**. tronscan.org 쓰면 Mainnet 필요.

## 1단계: TronLink + Nile

1. TronLink 열기
2. 네트워크 **Nile** 선택 (없으면 수동 추가: RPC `https://nile.trongrid.io`, Chain ID `3`)
3. nileex.io/join/getJoinPage 접속
4. **TronLink 주소 복사** → nileex에 **붙여넣기(입력)** → 제출 → TRX 2000 받기
   - **저장(제출) 안 되면 배포도 안 됨**

---

## 2단계: Nile에서 배포

### 방법 1: Remixd (저장 가능, 추천)

**상세:** `Remixd_로컬폴더_연결_방법.md`

1. 터미널: `npm install -g @remix-project/remixd`
2. 터미널: `remixd -s C:\Users\hosuk\link-directory -u https://www.tronide.io` (실행 상태 유지, 주소창 URL과 일치)
3. TronIDE → PLUGIN MANAGER → REMIXD → Activate
4. FILE EXPLORERS → **Connect to a local Plugin** → Connect
5. localhost → LinkUSD_TRON.sol 열기 → 수정 → **Ctrl+S** (로컬 저장됨)
6. Compile → Deploy

### 방법 2: Load a local file (저장 불가)

1. TronIDE 접속
2. FILE EXPLORER → **Load a local file** → LinkUSD_TRON.sol 선택
3. **4_Trc10.sol 무시.** Compile·배포는 **LinkUSD_TRON.sol**만.
4. Solidity Compiler → Compile → Deploy & Run → LinkUSD → Deploy

**"다른 이름으로 저장" 창** = .sol 아님. **취소** 클릭.

## 3단계: 전송 테스트

1. TronLink에서 토큰 추가 (주소 입력)
2. 전송 → 다른 주소로 보내기

**"의심스러운" 경고** = Nile 테스트넷에서 흔함. 기능에는 영향 없음. `Nile_의심표시_설명.md` 참고.

## 저장·컴파일·배포 구분

| 단계 | 의미 | 컨트랙트 주소 |
|------|------|---------------|
| **컴파일** | Solidity Compiler → Compile. artifacts에 LinkUSD.json 생성 | 아직 없음 |
| **배포** | Deploy 버튼 클릭 → TronLink 서명 | **이때** Deployed Contracts에 주소 나옴 |

**"compilerMetadata is modifying /artifacts/LinkUSD.json"** = 컴파일됨. 저장 아님. 배포 전.

**컨트랙트 주소** = **Deploy** 클릭하고 TronLink 서명한 **후**에 나옴.
