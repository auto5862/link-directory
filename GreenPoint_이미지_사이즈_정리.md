# Green Point — 이미지 사이즈 정리 (배포 전 준비)

**배포 전에 미리 준비. 사이즈 맞춰서.**

---

## 1. TronScan Token Logo

| 항목 | 값 |
|------|-----|
| **파일** | `GreenPoint_logo_100x100.png` |
| **사이즈** | 100×100 픽셀 |
| **용량** | 200KB 이하 |
| **형식** | PNG |
| **사용처** | TronScan Update Token Basic Info → Token Logo |

---

## 2. 텔레그램 채널 이미지

| 항목 | 값 |
|------|-----|
| **프로필 사진** | 512×512 픽셀, PNG/JPEG, 5MB 이하. 원형 크롭됨 → 중앙 배치 |
| **채널 배너** (선택) | 1280×640 픽셀, PNG/JPEG, 5MB 이하 |
| **파일** | `GreenPoint_logo_512x512.png`, `GreenPoint_banner_1280x640.png` |
| **상세** | `GreenPoint_텔레그램_채널_만들기.md` 참고 |

---

## 3. Sun Pump 이미지 (선택)

| 항목 | 값 |

## 4. 준비된 파일

| 파일 | 상태 |
|------|------|
| GreenPoint_logo_100x100.png | ✓ (100×100, TronScan용) |
