# 배포 전에 확인

## TronLink에서 할 일 (먼저)

1. **Nile 네트워크** 선택
   - Nile이 안 보이면 → 설정 → 네트워크 → Nile 추가
   - RPC: `https://nile.trongrid.io` / Chain ID: `3`
2. **nileex.io** 접속 → TronLink **주소 입력** (복사해서 붙여넣기) → TRX 2000 받기
   - **저장(제출) 안 되면 배포도 안 됨** — TRX 없으면 배포 불가
3. **TronIDE**에서 TronLink 연결 버튼 클릭 → 연결됐는지 확인

## TronIDE에서 할 일

**.sol 넣는 곳 = 하나.** Load a local file → LinkUSD_TRON.sol → Compile → Deploy

**4_Trc10.sol** = TronIDE 기본 파일. 무시. LinkUSD 코드 넣지 말고 **LinkUSD_TRON.sol**만 사용.

"다른 이름으로 저장" 창 = .sol 아님. 취소.

## Tronscan (배포 전 필수)

**지갑 주소 공개 태그 검증** — 배포에 쓸 지갑을 "이 프로젝트 소유"로 등록.

1. **nile.tronscan.org**/#/tools/verify-sign 접속 (Nile 테스트 시)
   - 메인넷이면 **tronscan.org**/#/tools/verify-sign
2. **Sign Message** 탭 → TronLink 연결 → 메시지 `I confirm this address belongs to Link USD.` → Sign → **서명 해시** 복사
3. **Verify Signature** 탭 → Address, Message, Signature 입력 → Verify → ✅ 확인
4. **nile.tronscan.org**/#/tools/contactUs (또는 tronscan.org) → **태그 제출** 선택 → 위에서 복사한 걸 붙여넣기 → 제출

**지갑·Tronscan 네트워크 일치:** Nile 테스트 → nile.tronscan.org + TronLink Nile. 메인넷 → tronscan.org + TronLink Mainnet.

## 저장 vs 컴파일 vs 배포

- **컴파일** = Compile 클릭. artifacts 수정됨. 주소 없음.
- **배포** = Deploy 클릭 → TronLink 서명. **이때** 컨트랙트 주소 나옴.
