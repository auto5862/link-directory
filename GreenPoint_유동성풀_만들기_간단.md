# TronScan GRP 유동성 풀 만들기 (간단)

---

## 1. 접속

```
https://sunswap.com/#/add
```

→ **Connect Wallet** → TronLink 연결

---

## 2. 토큰 선택

| 입력란 | 입력 |
|--------|------|
| **첫 번째** | **TRX** 클릭해서 선택 |
| **두 번째** | `TPG7971YWN5kpMBwdzLbtMpEwJNMkh9Ux6` 붙여넣기 (또는 **Manage** → **Import** → 주소 입력) |

→ Green Point (GRP) 선택

---

## 3. 금액 입력

| 토큰 | 예시 |
|------|------|
| **TRX** | 100 (또는 원하는 양) |
| **GRP** | 자동 계산됨. 또는 직접 입력 (예: 1,000,000) |

→ **초기 가격** = TRX : GRP 비율

---

## 4. 풀 생성

1. **Create Pool** (또는 **Supply**) 클릭
2. TronLink → **확인** (첫 토큰 승인)
3. 다시 **확인** (두 번째 토큰 승인)
4. **Supply** 클릭 → TronLink 최종 확인

---

## 5. 완료

- GRP/TRX 풀 생성됨
- SunSwap에서 거래 가능
- 거래 수수료 일부 수익

---

- TronScan GRP (지갑에 있음)
- TRX (풀용 + 가스비)
