# 문서 10개 — 어디에 어떤 식으로 올리는지

**한 번에 10개 올리는 곳 없음. 카테고리마다 제출처가 다름.**

---

## 각 카테고리마다 하나씩 (제출처별)

| # | 문서 | 제출처 | 방법 |
|---|------|--------|------|
| 1 | **백서** | GitHub 저장소에 올림 | 파일 올리면 → GitHub Pages URL 자동 |
| 2 | **Record Token** | tronscan.org → Tokens → Record Token | 폼에 주소, 로고 URL 입력 → Submit |
| 3 | **소스 검증** | tronscan.org/#/contracts/verify | 주소 입력 + .sol 파일 **업로드** → Verify |
| 4 | **토큰 평판** | tronscan.org → 문의하기 → 토큰 평판 | 폼에 URL, 이메일 등 입력 → 제출 |
| 5 | **문의하기 (태그)** | tronscan.org → 문의하기 → 기타 | 폼에 내용 입력 → 제출 |
| 6 | **support** | support.tronscan.org | 새 티켓 → 제목·본문 입력 → Submit |
| 7 | **포럼** | forum.trondao.org | 새 글 작성 → 제목·본문 입력 → 게시 |
| 8 | **Audit** | GitHub 저장소에 올림 | 파일 올리면 → URL로 참조 |
| 9 | **지갑 검증** | tronscan.org/#/tools/verify-sign | 주소 입력 + 서명 → 검증 |
| 10 | **API** | (없음) | — |

---

## 정리

| 질문 | 답 |
|------|-----|
| **한 번에 10개 올리는 곳?** | 없음 |
| **각각 카테고리마다 하나씩?** | ✓ 맞음. 제출처가 9곳 (API 제외) |
| **방법** | 카테고리별로 해당 사이트 가서 폼 작성 또는 파일 업로드 |

---

| 순서 | 사이트 | 경로 |
|------|--------|------|
| 1 | tronscan.org | Tokens → Record Token |
| 2 | tronscan.org | #/contracts/verify |
| 3 | tronscan.org | 문의하기 → 토큰 평판 |
| 4 | tronscan.org | 문의하기 → 기타 |
| 5 | tronscan.org | #/tools/verify-sign |
| 6 | support.tronscan.org | Submit a request |
| 7 | forum.trondao.org | 새 글 |
| 8 | github.com | 저장소에 파일 push |
| 9 | (API) | 없음 |
