# C. Base 메인넷 (MetaMask) — 실제, ETH 필요

## C-1. 배포 전 (순서대로)

| # | 할 일 | 완료 |
|---|-------|------|
| 1 | chainlist.org 접속 → "테스트넷 포함" **해제** | ☐ |
| 2 | "Base" 검색 → 메타마스크에 추가 | ☐ |
| 3 | 거래소에서 ETH 0.005~0.01 구매 (약 $15~50) | ☐ |
| 4 | 출금 → 네트워크 **Base** 선택 (이더리움 아님) | ☐ |
| 5 | MetaMask 주소로 출금 | ☐ |
| 6 | MetaMask에서 Base 선택 → ETH 잔액 확인 | ☐ |
| 7 | remix.ethereum.org 접속 | ☐ |
| 8 | LinkUSD_Ethereum.sol 파일 생성 → 코드 붙여넣기 → 저장 | ☐ |
| 9 | Solidity Compiler → Compile → 녹색 체크 확인 | ☐ |
| 10 | Deploy & Run → Environment: Injected Provider - MetaMask | ☐ |
| 11 | MetaMask 네트워크가 **Base**인지 확인 | ☐ |

## C-2. 배포

| # | 할 일 | 완료 |
|---|-------|------|
| 12 | Contract: LinkUSD 선택 | ☐ |
| 13 | Deploy 클릭 → MetaMask 서명 | ☐ |
| 14 | deployed contracts에서 주소(0x...) 복사 → **메모장 저장** | ☐ |

## C-3. 배포 후

| # | 할 일 | 완료 |
|---|-------|------|
| 15 | app.uniswap.org 접속 | ☐ |
| 16 | MetaMask 연결 → 네트워크 **Base** 확인 | ☐ |
| 17 | Pool → New position | ☐ |
| 18 | Select a token → 컨트랙트 주소 붙여넣기 → Import | ☐ |
| 19 | 토큰 1: ETH, 토큰 2: Link USD | ☐ |
| 20 | 금액 입력 (ETH + LUSD) | ☐ |
| 21 | Approve → MetaMask 서명 | ☐ |
| 22 | Add / Supply → MetaMask 서명 | ☐ |
| 23 | 스왑 가능·시세 표시 확인 | ☐ |
| 24 | MetaMask에 토큰 추가 (선택): 토큰 가져오기 → 컨트랙트 주소 입력 | ☐ |
