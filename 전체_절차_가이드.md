## Part A. TRONSCAN 토큰 평판 시스템 (필수 숙지)

TRONSCAN 커뮤니티 피드백 반영, 토큰 평판 등급 개정. **"중립" 이상 등급**을 받으려면 아래를 충족해야 함.

| 번호 | 요건 | 내용 |
|------|------|------|
| 1 | 필수 정보 | 공식 웹사이트, **로고**, 공식 연락처 이메일, **백서 URL**, 최소 1개 이상 **소셜 미디어 링크** |
| 2 | 소스 검증 | 계약 소스 코드 **검증 완료**. 특별한 사유로 검증 불가 시 TRONSCAN에 연락 |
| 3 | 팀 공개 | 설립자·팀 구성원 공개 프로필이 **공식 웹사이트**에 공개 |
| 4 | 소통 채널 | 소셜 미디어 또는 커뮤니티 소통 채널 **활발히 유지** |
| 5 | 신뢰 등급 | "안전하지 않음" 또는 "사기"로 표시되지 않음 |

**토큰 등급 업데이트 신청:** TRONSCAN → 문의하기 → 토큰 등급 업데이트

## Part C. 정확한 흐름 (우리 USDT → TRX → XRP/ETH)

### 전체 경로

```
[1] 우리 USDT 전송 (지갑 A → 지갑 B)
        ↓
[2] 상대: token-receive.html로 수신 확인
        ↓
[3] 상대: SunSwap에서 우리 USDT → TRX 스왑  ← 반드시 TRX로 먼저
        ↓
[4] 상대: TRX → 거래소 입금
        ↓
[5] 상대: 거래소에서 TRX → XRP(또는 ETH) 교환
        ↓
[6] 상대: XRP 출금 (선택)
```

---

### 부연 설명

#### Q1. 우리 USDT를 거래소에 바로 입금할 수 있나?

**아니오.** 거래소는 **공식 USDT** 등 특정 컨트랙트 주소만 지원. 우리 토큰은 미지원. 직접 입금 불가.

#### Q2. 왜 반드시 TRX로 스왑해야 하나?

- 우리 USDT는 **거래소 미지원**
- TRX는 TRON 네이티브 토큰으로 **거래소 대부분 지원**
- 따라서 **우리 USDT → TRX → 거래소** 순서가 필수

#### Q3. 스왑은 어디서 하나?

**SunSwap** (https://sunswap.com). TronLink는 지갑일 뿐, 스왑 기능 없음. SunSwap에 TronLink 연결 후 스왑.

#### Q4. TRX 수수료가 발생하는 구간은?

| 구간 | TRX 수수료 |
|------|------------|
| SunSwap 스왑 (우리 USDT → TRX) | ✅ 에너지/대역폭 |
| TRX → 거래소 전송 | ✅ 전송 수수료 |
| 거래소 내 TRX→XRP 교환 | ❌ (거래소 수수료만) |

**상대 지갑에 TRX가 없으면** 스왑 트랜잭션 실행 불가. USDT와 함께 **소량 TRX**(10~20)를 같이 보내거나, 상대가 미리 TRX 보유.

#### Q5. 크로스체인 브릿지가 필요한가?

**아니오.** 우리 흐름은 TRON 내 스왑 → 거래소 입금 → 거래소 내 교환. 브릿지 불필요.

## Part D. 단계별 상세 절차

### D-1. 우리 USDT 전송 (보내는 쪽)

1. token-receive.html 또는 TronLink
2. 받을 주소, 금액 입력
3. transfer() 실행 → Sign

### D-2. 수신 확인 (받는 쪽)

1. token-receive.html 접속
2. 지갑 연결 또는 TronScan 주소 검색
3. 잔액 확인 (Custom Token 추가 불필요)

### D-3. 우리 USDT → TRX 스왑 (받는 쪽)

1. SunSwap 접속, TronLink 연결
2. **Swap** 선택
3. From: 우리 USDT (컨트랙트 주소로 추가)
4. To: TRX
5. 수량 입력 → Swap → Sign

### D-4. TRX → 거래소 입금

1. 거래소 TRX 입금 주소 복사 (TRON 네트워크)
2. TronLink에서 TRX 전송
3. 입금 확인 대기

### D-5. 거래소에서 XRP/ETH 교환

1. TRX → XRP (또는 ETH) 거래
2. 출금 주소 입력 후 출금 (선택)
