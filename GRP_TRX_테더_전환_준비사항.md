## 2. 준비사항 체크리스트

### 공통
- [ ] **TronLink** 설치·연결
- [ ] **지갑에 GRP** 보유 확인 (TronScan 또는 Sun Pump)
- [ ] **TRX** 일부 보유 (가스비용, 최소 10~50 TRX 권장)

### TronScan GRP → TRX 스왑 시
- [ ] TRX/GRP 풀 **이미 있음** (5 TRX + 500 GRP 풀 생성됨) → 바로 스왑 가능
- [ ] TRX (Energy/대역폭용) — 풀 있으면 스왑만 하면 됨
- [ ] TronLink에 GRP 토큰 추가: `TPG7971YWN5kpMBwdzLbtMpEwJNMkh9Ux6`

### TronScan GRP → 공식 USDT 시 (2단계)
- [ ] 1단계: GRP → TRX (SunSwap)
- [ ] 2단계: TRX → 공식 USDT (SunSwap)

### Sun Pump GRP 판매 시
- [ ] Sun Pump 접속: https://sunpump.meme
- [ ] GRP 토큰 페이지에서 **판매** 버튼
- [ ] 구매자가 있으면 TRX로 받음 (시총 $69,420 도달 전까지는 Sun Pump 내에서만)

### 풀 없을 때 (풀 생성 필요)
- [ ] **TRX 700 이상** 또는 **Energy 대여** (tronpoolenergy.com 등)
- [ ] SunSwap: https://sunswap.com/#/add
- [ ] TRX + GRP(`TPG7971YWN5kpMBwdzLbtMpEwJNMkh9Ux6`) 입력 → Supply

## 3. 실행 순서 (가장 빠른 경로)

### A. TronScan GRP 보유 시 (풀 이미 있음)

1. **SunSwap** 접속: https://sunswap.com
2. **Swap** 탭
3. **From:** GRP (`TPG7971YWN5kpMBwdzLbtMpEwJNMkh9Ux6`) — Manage → Import 후 주소 입력
4. **To:** TRX
5. 금액 입력 → **Swap** → TronLink 승인

### B. TRX → 공식 USDT까지 (테더로 만들기)

1. 위 A 완료 후 TRX 보유
2. SunSwap **Swap** 탭
3. **From:** TRX
4. **To:** USDT (`TR7NHqjeKQxGTCi8q8ZY4pL8otSzgjLj6t`)
5. 금액 입력 → **Swap** → TronLink 승인

### C. Sun Pump GRP 보유 시

1. https://sunpump.meme 접속
2. GRP 토큰 페이지에서 **판매** 버튼
3. TRX로 판매 (구매자 있을 때)
