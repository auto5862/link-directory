## 5. Sun Pump Launch 순서

### 5-1. 접속

**URL:** https://sunpump.meme

1. 접속
2. **Connect Wallet** → TronLink 연결

### 5-2. Launch

1. 상단 **Launch** 클릭
2. 폼에 아래 값 입력

### 5-3. 기입란

| 기입란 | 입력값 |
|--------|--------|
| Token Name | `USDT (TeatherUSD)` |
| Ticker Symbol | `USDT` |
| Description | `USDT (TeatherUSD) — TRON TRC20. Sun Pump 신뢰 쌓고 TRON으로.` |
| Website | `https://auto5862.github.io/link-directory/usdt-sunpump.html` |
| Telegram | (채널 만들고 URL 입력) |
| Twitter | (선택) |
| Image | (선택) 지갑 USDT 아이콘과 동일 또는 100×100 PNG |

### 5-4. 실행

1. **Launch** 클릭
2. TronLink → ~20 TRX 결제 → **Confirm**
3. 완료 → 바로 거래 가능
