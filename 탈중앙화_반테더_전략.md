# 탈중앙화 & 반테더(Alternative) 전략

---

## 1. 탈중앙화란?

**중앙 권한 없이 동작하는 구조**

| 요소 | 중앙화 | 탈중앙화 |
|------|--------|----------|
| **관리** | 한 기관이 통제 | 스마트 컨트랙트·다수 참여 |
| **검증** | 기관이 검증 | 온체인·다수 노드 |
| **실행** | 서버 의존 | 블록체인 상 실행 |

**USDT(테더)의 한계:** 테더 회사가 발행·예비자산 관리. 사실상 **중앙화**에 가깝다.

---

## 2. 우리 LUSD의 탈중앙화 요소

| 항목 | 설명 |
|------|------|
| **온체인** | 전송·burn·lock 모두 블록체인에 기록 |
| **자체 보관** | 사용자가 지갑으로 직접 보유 |
| **DEX 상장** | SunSwap 풀 = 중앙 거래소 없이 스왑 |
| **제한** | owner가 lock 가능 → 일부 중앙화 |

---

## 3. 반테더(Alternative) 포지셔닝

**테더를 대체하는 게 아니라, 다른 선택지를 제공**

| 테더 | 우리 LUSD |
|------|-----------|
| 시장 1위 | 대안 |
| 회사 통제 | 온체인 Burn·Lock |
| 예비자산 논란 | 투명한 공급량 |

**나인봇 vs 세그웨이, 샤오미 vs 애플**처럼 — 자리는 바뀔 수 있다.

---

## 4. 각 코인의 역할

- **테더:** 기존 시장 점유
- **LUSD:** Burn·Lock·투명성
- **GRP:** 커뮤니티·유틸리티

각자 다른 **능력치**로 다른 자리를 노린다.
