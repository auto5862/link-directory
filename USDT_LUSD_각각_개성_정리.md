# USDT vs LUSD — 각각의 개성

**USDT 하나, LUSD 하나. 서로 다른 역할.**

---

## 1. 공식 USDT (테더)

| 개성 | 설명 |
|------|------|
| **역할** | 시장 1위 스테이블코인 |
| **특징** | 회사 발행, 예비자산, 거래소·결제에 널리 쓰임 |
| **우리 할 일** | 없음. 테더가 만든 토큰. 우리는 **풀에 넣을 때만** 사용 |

---

## 2. 우리 LUSD (Link USD)

| 개성 | 설명 |
|------|------|
| **역할** | 테더의 **대안** — Burn·Lock·투명성 |
| **특징** | 온체인 burn, 계정 lock, 공급량·거래 추적 가능 |
| **우리 할 일** | DEX 상장, 커뮤니티, 사용처 확대 |

---

## 3. 각각 어떻게 쓸지

| 토큰 | 용도 |
|------|------|
| **USDT** | 기존 생태계·거래소·결제 (이미 쓰이는 곳) |
| **LUSD** | Burn·Lock이 필요한 곳, 우리 생태계, 대안 스테이블코인 |

---

## 4. DEX 풀에서의 관계

| 풀 | 의미 |
|-----|------|
| **LUSD/TRX** | LUSD를 TRX로 스왑 |
| **LUSD/USDT** | LUSD ↔ USDT 교환 (테더와 연결) |

---

**정리:** USDT = 시장 표준. LUSD = Burn·Lock 있는 대안. 각자 개성에 맞게 쓰면 됨.
