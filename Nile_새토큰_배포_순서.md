# Nile 새 토큰 배포 — 의심 안 뜨게

**이름:** Blue Token / BLU — USD·Tether·USDT 전부 제거. 의심 플래그 회피용.

---

## 순서

1. TronIDE 접속
2. **Load a local file** → **BlueToken_TRON.sol** 선택
3. Solidity Compiler → Compile
4. Deploy & Run → **BlueToken** 선택 → Deploy
5. TronLink 서명
6. 컨트랙트 주소 복사
7. nile.tronscan.org에서 **의심 표시 안 뜨는지** 확인
