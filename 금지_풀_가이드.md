# 금지·경고·풀(해제) — 한 번에 정리

**배포 전에 꼭 읽고 지키기. 경고 먹으면 여기서 풀 방법 확인.**

**통합 문서:** `배포_전_후_토큰등급_전부_업데이트.md` (배포 전·후·토큰 등급·링크 전부)

---

## 1. 금지 — 절대 하지 말 것

| 금지 항목 | 이유 |
|-----------|------|
| **Tether, USDT, TrueUSD, TUSD** 이름·심볼 사용 | TronScan 자동 사기로 표시 → 경고·차단 |
| **Contract Name = TetherUSD** | TronScan Overview에 고정. 블랙리스트/피싱 경고의 핵심 원인. 배포 후 수정 불가. |
| **TronIDE에 예전 코드 붙여넣기** | TetherUSD, Test USD, TUSD 들어 있으면 → 사기로 뜸 |
| **배포 전 검증 안 하고 배포** | 지갑 검증 안 되어 있으면 토큰 신뢰도 낮음 |

---

## 2. TronIDE 배포 시 꼭 확인할 것

**배포 전에 코드 3줄 확인:**

```
contract LinkUSD {           ← 이거여야 함 (TetherUSD 아님)
    string public name = "Link USD";   ← Link USD (Test USD 아님)
    string public symbol = "LUSD";     ← LUSD (TUSD 아님)
```

- **파일:** `LinkUSD_TRON.sol` (이 프로젝트 폴더에 있는 것)
- **붙여넣기:** 이 파일 **전체**를 복사해서 TronIDE 1_Storage.sol에 덮어쓰기
- **CONTRACT 선택:** Deploy 시 **LinkUSD** 선택 (TetherUSD 아님)

---

## 3. 경고 뜨면 — 풀(해제) 방법

### 3-1. 토큰 평판 업데이트 (가장 먼저)

1. https://tronscan.org/#/tools/contactUs?feedbackType=tokenLevel 접속
2. 또는 토큰 페이지 경고 메시지에서 **"토큰 평판 업데이트"** 클릭
3. 제출: 웹사이트 URL, 백서 URL, 이메일, 소셜 1개 이상
4. **기입란:** `토큰_평판_업데이트_기입란_전부.md` | **참조 URL:** `참조_URL_한번에_복사.md`

### 3-2. 태그 제거 요청

- tronscan.org → **문의하기** → **태그 제거**
- TronLink 발행자 주소 연결
- "Link USD 프로젝트, Tether/USDT 아님" 명시

### 3-3. 문제 및 제안

- **문의하기** → **문제 및 제안**
- "프로젝트 테스트, 사기 아님" + GitHub·백서 링크 제출

### 3-4. 오탐 해제

- **문의하기** → **기타** → 오탐 해제 요청
- 제출 자료: GitHub, 백서, 웹사이트, 자체 보안 검토

---

## 4. 풀 안 되면 — 마지막 수단

- **새 이름·심볼**로 수정 후 TronIDE에서 **새로 배포**
- 기존 컨트랙트는 수정 불가 → 새 주소 생김
- 배포 TRX 다시 소모
