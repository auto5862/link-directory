# 소스 코드 검증 — 화면 기입란 (한눈에 보기)

**URL:** https://tronscan.org/#/contracts/verify

---

## ⚠️ 소스 검증 받으면 바뀌는 것 / 안 바뀌는 것

| 항목 | 검증 후 바뀜? |
|------|---------------|
| **Contract Name** (TetherUSD 등) | ❌ **안 바뀜** — 배포 시 이미 고정됨 |
| **피싱 경고** | ❌ **안 바뀜** — Contract Name이 TetherUSD면 계속 사기 표시 |
| **"계약 코드가 확인되지 않았습니다"** | ✅ **바뀜** — 검증 완료 시 사라짐 |
| **소스 코드 공개** | ✅ **바뀜** — TronScan에서 코드 보임 |

→ **검증은 "코드 공개 + 미검증 문구 제거"만 함. Contract Name·피싱 경고는 그대로.**

## 기입란 요약표

| 기입란 | 뭐 넣나 | 예시 |
|--------|---------|------|
| **Contract Address** | 배포한 컨트랙트 주소 | TNXDJXHKBXZFWLksXpvccy9oKZNw9kHPVS |
| **Solidity Compiler Version** | 배포 시 사용한 컴파일러 | v0.8.25 또는 tron_v0.8.25 |
| **Optimization** | 최적화 여부 | No (TronIDE 기본) |
| **Runs** | 최적화 횟수 | 0 |
| **License** | 라이선스 | None 또는 MIT |
| **VM version?** | TRON VM 버전 | default |
| **ViaIR?** | IR 컴파일 사용 여부 | No |
| **Main Contract** | contract 이름 | TetherUSD 또는 LinkUSD |
| **Upload Contract File(s)** | .sol 파일 | TetherUSD.sol 또는 LinkUSD_TRON.sol |

## 소스코드 "적는" 곳 = Upload Contract File(s)

- **직접 타이핑하는 칸 아님**
- **파일 업로드**로 올리는 것
- "계약서 파일을 업로드하세요" / "Upload Contract File(s)" 클릭 → `.sol` 파일 선택 → 업로드
