## TronIDE에서 배포 시 사용한 설정 확인

TronIDE (tronide.io)에서 배포했으면:

1. TronIDE 접속 → **설정(톱니바퀴)** 또는 **Compile** 탭
2. **Compiler version** 확인
3. **Optimization** 체크 여부 확인
4. 위 값과 **동일하게** 검증 폼에 입력

## 그래도 안 되면

- TronScan **문의하기** → "Contract verification failed for TNXDJXHKBXZFWLksXpvccy9oKZNw9kHPVS, need correct compiler parameters"
- 또는 TronIDE **배포 당시 URL**에 `optimize=true/false` 등 파라미터가 있었는지 확인
