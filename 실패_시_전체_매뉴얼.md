# 실패 시 전체 매뉴얼 — 뭐가 안 되면 이거 보기

**어디서 막혔는지 → 아래에서 해당 항목 찾기.**

---

# 1. 배포 실패

| 증상 | 원인 | 해결 |
|------|------|------|
| Deploy 클릭해도 안 됨 | TronLink 연결 안 됨 | TronIDE에서 TronLink 연결 확인 |
| "Insufficient balance" | TRX 부족 | Nile: nileex.io에서 TRX 받기 / Mainnet: 거래소에서 TRX 구매 |
| Energy 0/0 | 에너지 부족 | TRX 스테이킹 또는 Energy 대여 |
| CONTRACT 안 뜸 | 컴파일 안 함 | Solidity Compiler → Compile 먼저 |
| 메인넷에 배포됨 | Nile 선택 안 함 | TronLink에서 네트워크 **Nile** 확인 |

---

# 2. 소스 코드 검증 실패

| 증상 | 원인 | 해결 |
|------|------|------|
| "verification failed" | 파라미터 불일치 | `Verify_실패_해결.md` — Optimization, Runs, Compiler 바꿔서 재시도 |
| "계약 코드가 확인되지 않았습니다" | .sol 파일 안 올림 | **Upload Contract File(s)** → .sol 전체 업로드 |
| 바이트코드 불일치 | 배포 후 .sol 수정함 | 수정 전 원본으로 다시 업로드 |

**재시도 순서:** `소스검증_실패_재시도_순서.md`

---

# 3. SunSwap — 토큰 안 뜸

| 증상 | 원인 | 해결 |
|------|------|------|
| 토큰 목록에 없음 | 유동성 풀 없음 | **직접 링크** 사용: `https://sunswap.com/?tokenAddress=[주소]&type=add` |
| Nile에서 안 뜸 | SunSwap Nile 미지원 | 메인넷에서만 풀 생성 가능 |

---

# 4. TRX / Energy 부족

| 증상 | 해결 |
|------|------|
| TRX 0 | Nile: nileex.io / Mainnet: 거래소 구매 |
| Energy 0/0 | TRX 스테이킹 또는 Energy 대여 |
| 전송·스왑 실패 | TRX 충전 후 재시도 |

**상세:** `현재_상황_복구_가이드.md`

---

# 5. TronIDE / Remixd

| 증상 | 해결 |
|------|------|
| "다른 이름으로 저장" 창 | .sol 아님. **취소** 클릭 |
| Remixd "path outside shared folder" | `Remixd_로컬폴더_연결_방법.md` 7번 — **Load a local file** 사용 |
| 저장 안 됨 | Load a local file은 저장 불가. Cursor에서 수정 후 다시 Load |

---

# 6. 사기/의심/블랙홀 표시

| 증상 | 해결 |
|------|------|
| 토큰에 "의심" 표시 | Nile: 흔함. 기능에는 영향 없음 |
| 메인넷 "피싱" 경고 | 문의하기 → 태그 제거 요청 / support.tronscan.org |
| Contract Name TetherUSD | 재배포 시 **LinkUSD**로 배포 |

---

# 7. Record Token / Update Token

| 증상 | 해결 |
|------|------|
| 입력 칸 안 뜸 | F5 새로고침, Ctrl+Shift+R 강력 새로고침 |
| 로고 크기 오류 | 100×100 PNG, 200KB 이하 (BlueToken_logo_100x100.png) |
