LinkUSD (LUSD) 받은 토큰 확인

토큰을 받았나요? 아래 방법 중 하나로 확인하세요.

컨트랙트 주소 (배포 후 입력)

① 지갑 연결해서 잔액 보기

TronLink 연결하면 이 페이지에서 바로 잔액 확인 (토큰 추가 안 해도 됨)

② TronScan에서 확인

주소 검색하면 모든 TRC20 내역 자동 표시. 토큰 추가 불필요.

Nile TronScan에서 보기 → Mainnet TronScan에서 보기 →

테스트넷=Nile / 메인넷=Mainnet

②-1 보내기 (토큰 추가 없이)

지갑 연결 → 받을 주소·금액 입력 → 이 페이지에서 바로 전송 (Custom Token 추가 불필요)

③ TronLink 지갑에 토큰 추가

버튼 한 번으로 지갑에 표시 (이후 지갑에서 바로 보임)

④ CoinMarketCap 등록 (장기)

등록되면 TronLink에서 자동 표시될 수 있음. SunSwap 상장 + 거래량 필요.
