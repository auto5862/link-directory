# GitHub 저장소 — Create 시 폼 입력용 (제대로 꾸미기)

**Create a new repository** 폼에 그대로 복사해서 넣기.

---

## 1. Repository name
```
link-directory
```
(이미 입력됨. link-directory is available.)

---

## 2. Description (0/350 — 여기 채우기)

**▼ 아래 중 하나 복사해서 Description 칸에 붙여넣기**

### 옵션 A (한글, 권장)
```
Link USD (LUSD) — TRON TRC20 스테이블코인. 온체인 Burn·Lock·투명성. 백서·도구·배포 가이드 포함. MIT.
```

### 옵션 B (영문, 검색·노출용)
```
Link USD (LUSD) — TRON TRC20 stablecoin. On-chain Burn, Lock, full transparency. Whitepaper, tools, deployment guide. MIT License.
```

### 옵션 C (한+영 혼합)
```
Link USD (LUSD) | TRON TRC20 stablecoin. On-chain Burn & Lock. Whitepaper, web tools, deployment guide. MIT.
```

---

## 3. Choose visibility
- **Public** 선택 (이미 선택됨)

---

## 4. Add a README file
- **체크 안 함** (로컬에 README.md 이미 있음 → 업로드할 때 포함)

---

## 5. Add .gitignore
- **None** (또는 비워둠)

---

## 6. Choose a license
- **MIT License** 선택 (코드에 SPDX MIT 명시됨)

---
