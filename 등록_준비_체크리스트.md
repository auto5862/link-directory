## 3. 소셜 미디어 (최소 1개 이상)

TRONSCAN "중립" 이상 등급 요건: **최소 1개 이상 소셜 미디어 링크** + **활발히 유지**

### 3-1. 소셜을 거의 안 하는 경우

| 옵션 | 난이도 | 설명 |
|------|--------|------|
| **GitHub** | 쉬움 | 프로젝트 저장소 URL만 있어도 됨. 이미 코드 있으면 그대로 링크 |
| **Telegram** | 쉬움 | 채널 생성 → 백서 링크·공지 1회 포스팅 → 링크 제출. 이후 가끔이라도 업데이트 |
| **Twitter(X)** | 보통 | 계정 생성 → 프로필에 백서 링크, 핀 고정 → 링크 제출 |

### 3-2. "가입해서 주면 될까?"

- **가능함.** 새로 가입해서 채널/계정 만들고 링크만 제출해도 "있음" 요건은 충족.
- **다만** "활발히 유지"는 채널이 존재하고, 필요 시 문의에 응할 수 있는 상태를 의미. 완전 방치보다는 가끔 확인하는 편이 안전.

### 3-3. 추천 조합

1. **GitHub** (저장소 URL) — 필수에 가깝게 사용
2. **Telegram** 또는 **Twitter** 중 1개 — 공지·문의용
