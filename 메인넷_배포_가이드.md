# 메인넷 배포 - 한 번에 정리

**배포 전:** `배포_전_할_일_순서.md` 먼저 진행  
**TronScan 공식:** `TronScan_공식_가이드_정리.md` (콜드 월렛, 계약 검증, 주소 태그 검증 등)

---

## ⚠️ 필수: 메인넷 TRX 350개 이상

- **실제 비용** (환불 불가)
- 거래소에서 TRX 구매 후 본인 지갑으로 전송

---

## 0단계: 메인 지갑 주소 공개 태그 검증 (계약 전 필수)

배포 전에 발행자 지갑을 TronScan에 검증해두면 토큰·계약 신뢰도가 올라감.

1. https://tronscan.org/#/tools/verify-sign → **Sign Message**
2. TronLink 연결 → 배포에 쓸 **메인 지갑** 선택
3. 메시지: `I confirm this address belongs to Link USD.`
4. 서명 → [문의하기](https://tronscan.org/#/tools/contactUs)로 서명 해시 + 메시지 제출

---

## 1단계: TronLink → Mainnet 전환

1. TronLink 클릭
2. 좌측 하단 **Nile** → **Mainnet** 클릭
3. 메인넷 TRX 잔액 확인 (350 이상)

---

## 2단계: TronIDE 배포 (계약서 작성·배포는 여기서)

1. https://www.tronide.io/ 접속
2. TronLink 연결 (**Mainnet**인지 확인)
3. 1_Storage.sol에 **TetherUSD_Pro.sol** 코드 붙여넣기 → Ctrl+S
4. 왼쪽 **번개(⚡)** → **Deploy**
5. ENVIRONMENT: **Injected TronWeb**
6. CONTRACT: **LinkUSD**
7. **Deploy** → TronLink **Sign**
8. 배포 완료 후 **컨트랙트 주소 복사** (T로 시작)

---

## 3단계: TronLink에 토큰 추가

1. TronLink → **Custom Token** (또는 토큰 추가)
2. 복사한 컨트랙트 주소 붙여넣기
3. 저장

---

## 4단계: TronScan 이미지 로고 등록 ⭐

1. **tronscan.org** 접속 (nile 아님!)
2. 상단 **Tokens** → **Record Token** 클릭
3. **TRC20** 선택
4. 컨트랙트 주소 입력
5. **Logo (imgUrl)** 칸에 입력:
   - **권장**: LinkUSD 전용 로고 256×256 PNG → imgur.com 또는 imgbb.com 업로드 → 이미지 주소 복사 → 붙여넣기
6. **Submit**

### 금액 현재가 (Current Price)
- 커스텀 토큰은 TronLink에서 **$0** 표시 (정상)
- DEX(SunSwap) 상장 시 가격 자동 반영
- 레포트: "미상장 토큰은 가격 피드 없어 $0 표시"로 설명

---

## 4-2단계: 소스 코드 검증 (계약 코드 표시) ⭐

**도구:** https://tronscan.org/#/contracts/verify

**입력값 (배포 시 설정과 정확히 일치):**
- **Runs**: 0 (최적화 안 썼으면)
- **Optimization**: No
- **License**: None 또는 MIT
- **Solidity Compiler Version**: 0.8.x (배포 시 사용한 버전)
- **Main Contract**: **LinkUSD**
- **Contract Address**: 배포한 컨트랙트 주소
- **Upload Contract File(s)** → TetherUSD_Pro.sol 업로드

---

## 5단계: 피싱 경고 피하기 (선택)

Nile에서 "Tether USD / USDT" 때문에 경고가 떴음. 메인넷에서도 같은 경고 가능.

**현재**: `name = "Link USD"`, `symbol = "LUSD"` (Tether/USDT/TrueUSD와 완전히 다름)

## 위험경고 해결 (기회 한 번)

**상황:** "검증되지 않았거나 사기성 토큰으로 보고됨" 표시

1. **먼저 시도:** TronScan 경고 메시지의 "토큰 평판 업데이트" 클릭 → 웹사이트·백서·소셜·이메일 제출
2. **안 되면:** 새 이름·심볼로 수정 후 TronIDE에서 **새로 배포** (기존 컨트랙트는 수정 불가)

**상세:** `전체_작업_기록.md` 참고

## 오탐 해제·"사기 아님" 각인

**위험 보고** 시: 증거 없으면 **오탐** 처리 → 태그 해지됨.

| 방법 | 경로 |
|------|------|
| **태그 제거** | 문의하기 → 태그 제거 → TronLink 발행자 주소 연결 |
| **문제 및 제안** | 문의하기 → 문제 및 제안 → "프로젝트 테스트, 사기 아님" 설명 |
| **기타** | 문의하기 → 기타 → 오탐 해제 요청 |

## 추가로 할 수 있는 것 전부

| 항목 | 절차 |
|------|------|
| **계약 감사 보고서** | 문의하기 → 계약 감사 보고서 제출 → TronLink 해당 주소 연결 → 컨트랙트 주소 입력. 전문 감사 PDF 또는 `LinkUSD_자체_보안_검토.md` 제출 |
| **프로젝트 제출** | 문의하기 → 프로젝트 제출 |
| **TronScan My Account** | 가입 후 비공개 계정 이름, 관심 목록, 개인 거래 메모 설정 |

**있는 게 없는 것보다 나음.** 할 수 있는 건 다 하기.

## 잘못된 전송·중복 배포

| 상황 | 가능 여부 |
|------|-----------|
| 잘못된 주소/금액으로 전송 | **수신자만** 돌려보낼 수 있음. TronScan은 취소·역전 불가 |
