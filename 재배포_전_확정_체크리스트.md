# 재배포 전 확정 체크리스트 — 중복 금지

**배포 전에 모든 게 확정되어야 함. 앞 배포(TetherUSD)와 겹치거나 중복되면 안 됨.**

---

## 1. 앞 배포 vs 재배포 — 겹치면 안 되는 것 (중복 금지)

| 항목 | ❌ 앞 배포 (TetherUSD) — 이렇게 하면 안 됨 | ✅ 재배포 (Link USD) — 이렇게 해야 함 |
|------|-------------------------------------------|--------------------------------------|
| **사용 파일** | TetherUSD.sol | **LinkUSD_TRON.sol** |
| **contract 이름** | `contract TetherUSD` | **`contract LinkUSD`** |
| **토큰 name** | "Tether USD" | **"Link USD"** |
| **토큰 symbol** | "USDT" | **"LUSD"** |
| **TronIDE CONTRACT 선택** | TetherUSD | **LinkUSD** |
| **소스 검증 Main Contract** | TetherUSD | **LinkUSD** |
| **소스 검증 업로드 파일** | TetherUSD.sol | **LinkUSD_TRON.sol** |

→ **위 7개 중 하나라도 앞 배포랑 같으면 사기 표시됨. 전부 다르게 해야 함.**

---

## 2. 배포 전 확정 — 반드시 확인 (체크 안 되면 배포 금지)

### 2-1. 파일·코드 (LinkUSD_TRON.sol만 사용)

- [ ] **열어본 파일:** `LinkUSD_TRON.sol` (TetherUSD.sol 아님)
- [ ] **4번째 줄:** `contract LinkUSD {` (TetherUSD 아님)
- [ ] **name:** `"Link USD"` (Tether USD, Test USD, TUSD 아님)
- [ ] **symbol:** `"LUSD"` (USDT, TUSD 아님)
- [ ] 코드 안에 **Tether, USDT, TrueUSD, TUSD** 단어 없음

### 2-2. TronIDE 붙여넣기

- [ ] **복사한 곳:** 이 프로젝트 폴더 `LinkUSD_TRON.sol` 전체
- [ ] **붙여넣기:** 1_Storage.sol에 덮어쓰기 (다른 파일·다른 곳에서 복사 금지)
- [ ] **Ctrl+S** 저장

### 2-3. TronIDE Deploy

- [ ] ENVIRONMENT: **Injected TronWeb**
- [ ] CONTRACT: **LinkUSD** (TetherUSD 절대 금지)
- [ ] Deploy 클릭 전 **CONTRACT 드롭다운 다시 확인** → LinkUSD인지

### 2-4. 배포 후 소스 검증 (새 주소 기준)

- [ ] Main Contract: **LinkUSD**
- [ ] 업로드: **LinkUSD_TRON.sol** (TetherUSD.sol 아님)

## 3. 재배포 시 정확한 순서 (한 번에)

| 순서 | 할 일 | 확인 |

---

## 4. 중복 체크 — 앞 배포와 같은 게 하나라도 있으면 배포 중단

| 체크 | 앞 배포 (금지) | 재배포 (필수) | 통과? |
|------|----------------|---------------|-------|
| 1 | TetherUSD.sol 사용 | LinkUSD_TRON.sol 사용 | ☐ |
| 2 | contract TetherUSD | contract LinkUSD | ☐ |
| 3 | name "Tether USD" | name "Link USD" | ☐ |
| 4 | symbol "USDT" | symbol "LUSD" | ☐ |
| 5 | CONTRACT: TetherUSD 선택 | CONTRACT: LinkUSD 선택 | ☐ |
| 6 | Main Contract: TetherUSD | Main Contract: LinkUSD | ☐ |
| 7 | TetherUSD.sol 업로드 | LinkUSD_TRON.sol 업로드 | ☐ |

→ **7개 전부 "재배포" 쪽이어야 함. 하나라도 "앞 배포" 쪽이면 배포하지 말 것.**

---

## 5. 요약

- **배포 전:** 위 2·3·4 전부 체크 완료 후에만 Deploy
- **중복 = 사기:** 앞 배포(TetherUSD)와 같은 파일·이름·선택 쓰면 또 사기 표시
- **확정 = 체크 전부 ✓:** 체크 안 된 게 있으면 배포 금지
