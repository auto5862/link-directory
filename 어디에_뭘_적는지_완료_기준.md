# 어디에 뭘 적는지 · 언제 끝나는지

**"적어두기"는 나중에 폼에 붙여넣을 때 쓰려고 메모해두라는 뜻.**

---

## 0. 메인 지갑 주소 검증 — 어디에 뭘 적나?

**맞다. 내 주소(지갑)를 "이 프로젝트 소유"로 불러달라고(등록해달라고) 하는 거다.**

### 하는 곳 (2단계)

| 단계 | 어디 | 뭘 함 |
|------|------|-------|
| 1 | https://tronscan.org/#/tools/verify-sign | **Sign Message** → TronLink 연결 → 메시지 `I confirm this address belongs to Link USD.` 입력 → 서명 → **서명 해시 + 원본 메시지** 복사 |
| 2 | https://tronscan.org/#/tools/contactUs | **문의하기** 접속 → **태그 제출** (또는 주소 공개 태그 검증 관련 탭) 선택 → 위에서 복사한 **서명 해시 + 원본 메시지** 붙여넣기 → 제출 |

**기입란별 상세:** `지갑_주소_검증_기입란.md`

## 1. Telegram (또는 Twitter) — 어디에 적나?

### 만드는 곳 (상세: `텔레그램_채널_만들기.md`)
- **Telegram**: **채널** 생성 (그룹도 가능) → 제목 `Link USD`, 주제/설명에 웹·백서 링크
- **Twitter**: 계정 생성 → 프로필에 웹사이트·백서 링크 넣기

### 적어두는 곳 (메모)
- `배포_전_할_일_순서.md` 2번의 `_______________________` 칸에 URL 적어두기
- 예: `https://t.me/linkusd` 또는 `https://twitter.com/linkusd`

### 실제로 입력하는 곳 (제출)
- **TronScan 토큰 평판 업데이트** (10단계)
- 경로: 토큰 페이지 → "토큰 평판 업데이트" 클릭  
  또는 tronscan.org → **문의하기** → **토큰 등급 업데이트**
- 폼에 **소셜 1개 이상** 칸에 Telegram 또는 Twitter URL 붙여넣기

## 3. 언제 끝나나? — 완료 기준

### 필수 끝 (이거 하면 됨)
**10단계 토큰 평판 업데이트**까지 하면 **필수 작업 끝**.

- 1~9단계: 배포 + TronScan 등록 + 검증
- 10단계: 토큰 평판 업데이트 → 웹사이트, 백서, 이메일, 소셜(Telegram/GitHub 등) 제출

### 선택 (있으면 좋음)
- 11: 계약 감사 보고서
- 12: 프로젝트 제출
- 13: TronScan My Account
