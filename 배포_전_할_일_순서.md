## 중요: 본 지갑 vs 발행 토큰

| 구분 | TronScan 표시 |
|------|---------------|
| **본 지갑 주소** | 문제 안 뜸 (지갑 검색 시) |
| **발행한 토큰 컨트랙트** | "사기/검증 안 됨" 뜸 |

→ **지갑이 문제 아님. 토큰 이름·심볼이 Tether/USDT와 비슷해서 토큰 페이지만 사기로 뜸.**  
→ 지갑 새로 만들어도, 같은 이름 쓰면 토큰 또 사기 뜸. **이름·심볼을 완전히 다르게** 해야 함.

### 3. 메인 지갑 주소 공개 태그 검증 ⭐ (계약 전 필수)

**배포에 쓸 지갑**을 TronScan에 "이 프로젝트 소유"로 먼저 검증해두면, 토큰·계약 페이지 신뢰도가 올라감.

- [ ] https://tronscan.org/#/tools/verify-sign 접속
- [ ] **Sign Message** 선택 → TronLink 연결
- [ ] 배포에 쓸 **메인 지갑 주소** 선택
- [ ] 메시지 입력: `I confirm this address belongs to Link USD.`
- [ ] **Sign Message** → 지갑에서 서명
- [ ] 생성된 **서명 해시 + 원본 메시지**를 [문의하기](https://tronscan.org/#/tools/contactUs)로 제출

**상세:** `TronScan_공식_가이드_정리.md` 참고
